# دليل ميزة سجل النشاطات (Activity Logs)

تمت إضافة ميزة تتبع نشاطات المستخدمين بالكامل إلى النظام.

## 1. ما يتم تسجيله
يقوم النظام تلقائياً بتسجيل العمليات التالية:
- **المنتجات**: إضافة، تعديل، حذف، تعديل الكمية.
- **الموردين**: إضافة، تعديل، حذف.
- **المستخدمين**: إضافة، تعديل، حذف، تغيير كلمة المرور.
- **تسجيل الدخول**: يتم تسجيل كل عملية دخول ناجحة.

## 2. كيفية عرض السجلات
1. اذهب إلى نافذة "التقارير" (Reports) من القائمة الرئيسية.
2. اختر تبويب "سجل النشاطات" (Activity Logs).
3. ستظهر لك لوحة تحكم تحتوي على:
   - **ملخصات رقمية**: إجمالي النشاطات، المستخدمين النشطين، عمليات الإضافة والتعديل.
   - **جدول التفاصيل**: يعرض "من، ماذا، التفاصيل، التاريخ" لكل عملية.
   - **رسم بياني**: يوضح توزيع النشاطات (إضافة vs تعديل vs حذف).
4. يمكنك تصفية السجلات حسب التاريخ.

## 3. معلومات تقنية
- يتم حفظ السجلات في جدول `ActivityLogs` في قاعدة البيانات.
- يتم ربط السجل بالمستخدم الذي قام بالعملية عبر `UserID`.
- يتم استخدام `AuditHelper` لضمان تسجيل تواريخ الإنشاء والتعديل في جميع الجداول.

## 4. استكشاف الأخطاء وإصلاحها
- إذا لم تظهر بيانات في التقرير، تأكد من تسجيل الدخول بمستخدم ثم القيام ببعض العمليات (إضافة منتج مثلاً).
- تأكد من أن ملف قاعدة البيانات `GroceryStore.db` محدث (سيقوم النظام بتحديثه تلقائياً عند التشغيل).
